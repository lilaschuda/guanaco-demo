# Example 01: Multi-DSL Coexistence Architecture

This module demonstrates the core migration story of Guanaco: **Coexistence**. 

You do not have to rewrite your entire legacy codebase overnight. Guanaco is designed to act as a non-invasive layer that sits directly on top of Apache Camel, allowing Spring XML definitions, traditional Java Fluent DSL paths, and modern type-safe Guanaco routes to run simultaneously side-by-side.
